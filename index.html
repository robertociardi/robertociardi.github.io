# My Projects

Here are described some of the project I've worked on during my University Career, either by myself or with some colleagues. Some others and further details can be found on my github page.

ROS robotic arm inspector

A ROS prototype of an arm for vehicles' inspection, thought in particular for the inspection of trains brake pads.

A Neural Network for color classification

An Artificial Neural Network developed with MATLAB Deep Learning toolbox, to recognize differences between similar colors, and solve problems for industrial printing. Recently a conference paper about this work was published by IEEE.

Solar Tracker System

An embedded system developed on STM32F407 discovery board. The system represents a mechanical sunflower, detecting and following a light source.

Wearable Fall Detector

A wearable fall detector useful for tracking elder people movement and allowing instantaneous notification in case of falls.

IoT Bike Sharing

A simulated Bike Sharing system developed using Contiki (along with Cooja), and based on OM2M.

A c++ threadpool

An implementation of a "fair" threadpool used in a Client-Server communication, to instantiate and execute jobs for clients.

Java FairLock

An implementation of a "fair" lock to synchronize communication between threads.

VHDL Linear Interpolator

VHDL implementation of a Linear Interpolator. Some useful analysis were made using Vivado tools and further work was made on MATLAB.
